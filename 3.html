메모리

메모리는 프로그램이나 데이터를 저장하기 위한 장치인데, 크게 주기억장치(primary memory 혹은 main memory), 보조기억장치(secondary memory), 그리고 캐쉬(cache)로 나눈다. 주기억장치는 전자적인 속도로 동작하는 빠른 메모리로써 프로그램들이 수행되는 동안에는 주기억장치에 위치되어야 한다. 주기억장치는 일반적으로 동적 임의 접근 메모리(DRAM: Dynamic Random Access Memory)을 사용하며 이는 많은 반도체 기억 소자(semiconductor storage cells)로 구성되어 있다. 각 소자(cell)은 하나의 비트(bit)에 대한 정보를 저장할 수 있으며 이 소자가 모여 하나의 단어(word)를 구성하는데 주기억장치는 한번의 읽기 혹은 쓰기 동작에 단어 단위로 수행될 수 있도록 구성되며, 특정 단어를 접근(access)하기 위하여 주소(address)를 사용한다. 보조 기억 장치는 주기억 장치의 높은 가격으로 인하여 빈번하게 혹은 당장 사용하지 않는 프로그램이나 데이터를 저장하기 위하여 사용되는 기억 장치로써 디스크 혹은 테이프 등이 있으며 가격이 저렴하지만 접근 속도가 느리다. 캐쉬(cache)는 주기억장치의 접근 속도를 더욱 빠르게 하기 위하여 소용량의 고속 메모리이며 주로 정적 임의 접근 메모리(SRAM: Static Random Access Memory)로 구성된다.
